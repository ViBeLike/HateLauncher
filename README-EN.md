## Installation

### Download

Go to [**Releases**](../../releases) and download:

| File | Description |
|------|-------------|
| `HyTaLauncher_Setup_x.x.x.exe` | Installer (recommended) |
| `HyTaLauncher_Portable_x.x.x.zip` | Portable version (no installation) |

---

### Installer

1. Download `HyTaLauncher_Setup_x.x.x.exe` from Releases
2. Run the installer
3. Launch HyTaLauncher from desktop or start menu
4. Enter your nickname and click Play!

---

### Portable

1. Download `HyTaLauncher_Portable_x.x.x.zip` from Releases
2. Extract to any folder
3. Run `HyTaLauncher.exe`

## Mods

HyTaLauncher includes CurseForge integration for managing Hytale mods:

- Browse popular mods
- Search by name
- Filter by category and version
- One-click install
- Check for updates
- Easy uninstall
- Modpack support (create, edit, export, import)

### Modpacks

You can create modpacks to organize different mod configurations:

1. Open Mods Manager
2. Click "Manage Modpacks"
3. Create a new modpack and select mods to include
4. Switch between modpacks from the main window

Modpacks can be exported and shared with friends!

## Troubleshooting

### Game won't start
1. Try running as Administrator (Settings > Administrator)
2. Check if antivirus is blocking the game
3. Enable verbose logging and check logs folder

### Download issues
1. Try using a VPN
2. Enable mirror server in settings
3. If SSL errors occur, enable SSL bypass in settings

### Corrupted files
1. Use the Reinstall button (üîÑ) in the main window
2. This will re-download the selected version
