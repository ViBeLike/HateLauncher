## Features / Возможности

- Auto-download and install game files
- Automatic Java Runtime installation
- Version selection and updates
- CurseForge mods integration (browse, install, update)
- Multi-language support (English, Русский)
- Modern dark theme with Cinzel font
- Hytale news feed

## Installation / Установка

### Download / Скачать

Go to [**Releases**](../../releases) and download:

| File | Description |
|------|-------------|
| `HyTaLauncher_Setup_x.x.x.exe` | Installer (recommended) |
| `HyTaLauncher_Portable_x.x.x.zip` | Portable version (no installation) |

---

### Installer / Установщик

**EN:**
1. Download `HyTaLauncher_Setup_x.x.x.exe` from Releases
2. Run the installer
3. Launch HyTaLauncher from desktop or start menu
4. Enter your nickname and click Play!

**RU:**
1. Скачайте `HyTaLauncher_Setup_x.x.x.exe` со страницы Releases
2. Запустите установщик
3. Запустите HyTaLauncher с рабочего стола или из меню Пуск
4. Введите никнейм и нажмите Играть!

---

### Portable / Портативная версия

**EN:**
1. Download `HyTaLauncher_Portable_x.x.x.zip` from Releases
2. Extract to any folder
3. Run `HyTaLauncher.exe`

**RU:**
1. Скачайте `HyTaLauncher_Portable_x.x.x.zip` со страницы Releases
2. Распакуйте в любую папку
3. Запустите `HyTaLauncher.exe`

## Mods / Моды

HyTaLauncher includes CurseForge integration for managing Hytale mods:

- Browse popular mods
- Search by name
- One-click install
- Check for updates
- Easy uninstall

Mods are installed to: `%AppData%\Hytale\install\release\package\game\latest\Client\UserData\Mods`
